Queiroz Engenharia - Site (estático)
===================================

Como rodar local:
- Abra o arquivo index.html no navegador, OU
- Use um servidor local (recomendado):

  # com Node (npx)
  npx serve .

Botões:
- Todos os CTAs de WhatsApp já abrem conversa com mensagem pronta.
- O QR do rodapé abre o Instagram.

Arquivos:
- assets/img/hero-bg.jpg (background do hero)
- assets/img/qr-instagram.png
- assets/js/main.js (UX: menu mobile, links ativos, FAQ, botões WA, toast copiar)
- assets/css/extras.css (overlays do hero e acessibilidade)
